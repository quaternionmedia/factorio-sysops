# Factorio: Sysadmin
## IT Infrastructure Addon Mod

> Add IT infrastructure to enhance your factory. Monitor, automate, and optimize - without replacing any vanilla content.

---

## Key Design Principle

**This mod ADDS to vanilla Factorio - it does NOT replace anything.**

- All vanilla items, recipes, and entities unchanged
- IT infrastructure runs alongside your factory
- Works with or without Space Age
- Compatible with most other mods

---

## What This Mod Adds

### Data Layer
Your factory operations generate **data**. Collect it with sensors, transport it through network cables, and process it with servers.

### Monitoring
See what's happening in your factory with dashboards, metrics, and alerts. Know before things break.

### Automation
Create rules that respond to factory conditions. Integrate with the circuit network for powerful automation.

### Efficiency Bonuses
Properly monitored and optimized factories get **speed bonuses**, netted
against a debt penalty into one continuous curve. Currently reachable: +2%
with sensor coverage, +5% with a dashboard terminal. The full +35% ceiling
needs entities (automation controllers, ML processors, orchestration
controllers, quantum links) that don't exist yet -- see
[05-MECHANICS](05-MECHANICS.md).

### New Challenges
Cyber threats attack your IT infrastructure. Build firewalls, IDS, and security operations centers to defend.

## Quick Start

### Early Game
1. Research **IT Basics** (red science only)
2. Craft **Data Sensors** and attach to assemblers
3. Connect sensors with **Network Cables**
4. Build a **Basic Server** to process data
5. Place a **Dashboard Terminal** to see metrics

### Mid Game
1. Research automation technologies
2. Build **Automation Controllers** with rules
3. Connect to circuit network via **Circuit Bridge**
4. Gain efficiency bonuses from IT coverage

### Late Game
1. Deploy **Data Centers** for scale
2. Add **ML Processors** for insights
3. Build **Orchestration Controllers** for +25% bonus
4. Defend against cyber threats

### Space Age (Optional)
1. Each planet has IT specialization
2. **Vulcanus**: GPU computing, ML training
3. **Gleba**: Biological cold storage
4. **Fulgora**: Edge computing, resilience
5. **Aquilo**: Quantum computing (endgame)

## FAQ

**Q: Do I need IT to play?**  
A: No. Small factories work fine without it. IT becomes valuable as you scale.

**Q: Will biters attack my servers?**  
A: No. Biters attack walls. Cyber threats attack IT. Two separate systems.

**Q: Does this work with Bob's/Angel's/K2/SE?**  
A: It should! IT infrastructure is independent of recipe changes.

**Q: Can I add this to an existing save?**  
A: Yes! No vanilla changes means safe mid-game installation.
