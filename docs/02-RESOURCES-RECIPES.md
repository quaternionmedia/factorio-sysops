# Factorio: Sysadmin - Resources & Recipes (Revised)

## Design Principle

IT resources are **crafted from vanilla materials**, not replacements. This creates demand for existing factory output and integrates naturally with progression.

---

## New Resource: Data

### Data Generation

Every vanilla entity generates data as a byproduct:

| Entity Type | Data Generated | Rate |
|-------------|----------------|------|
| Assembling Machine | Production Data | 1/craft |
| Inserter | Throughput Data | 1/10 moves |
| Belt (per tile) | Logistics Data | 1/100 items |
| Mining Drill | Extraction Data | 1/ore |
| Furnace | Processing Data | 1/smelt |
| Lab | Research Data | 1/science consumed |
| Roboport | Logistics Data | 1/robot action |
| Train | Transport Data | 1/station visit |
| Reactor | Power Data | 1/tick while active |

### Data Properties
- **Decays** if not collected within 60 seconds
- **Accumulates** as Technical Debt if collected but not processed
- **Transforms** into Insights when processed by servers
- **Enables** automation when Insights reach thresholds

### Data Items

```
RAW DATA PACKET
- Generated by: Factory operations
- Collected by: Data Sensors
- Transported via: Network Cables
- Stack size: 1000

PROCESSED DATA
- Created from: Raw Data Packets (server processing)
- Used for: Insights, Research, Automation
- Stack size: 500

INSIGHT
- Created from: Processed Data (ML processing)
- Used for: High-tier automation, Optimization
- Stack size: 100
```

## Resource Flow Diagram

```
VANILLA FACTORY
     │
     ▼ (generates)
 RAW DATA ──────► DECAY (if uncollected)
     │
     ▼ (collected by Data Sensors)
DATA PACKETS
     │
     ▼ (transported via Network Cables)
 SERVERS ────────► TECHNICAL DEBT (if backlogged)
     │
     ▼ (processed)
PROCESSED DATA
     │
     ├──► INSIGHTS (via ML Processor)
     │         │
     │         ▼
     │    OPTIMIZATION ──► Factory Efficiency Bonus
     │
     └──► AUTOMATION ──► Circuit Network Commands
               │
               ▼
          VANILLA ENTITIES (controlled)
```
